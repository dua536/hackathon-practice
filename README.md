🛒 Organic Supermarket Management System

A Python-based Organic Supermarket Application developed using Tkinter that simulates an online organic grocery store. The system allows customers to browse products, add items to a shopping cart, and place orders through an interactive Graphical User Interface (GUI).

The application also includes an Order Management System where all customer orders can be viewed and stored automatically.

🚀 Features

Product Browsing System – View a wide range of organic grocery and skincare products

Shopping Cart Management – Add items, increase or decrease quantity, and view subtotals

Inventory Control – Product stock automatically updates when items are added to the cart

Checkout System – Customers enter personal details before placing an order

Multiple Payment Options – Cash, Card, and Online payment methods

Automatic Order Number Generation – Every order receives a unique order ID

Order Storage – Orders are saved in a text file for record keeping

Orders Management System – Admin can view all previous orders

Fullscreen GUI Interface – Clean and user-friendly graphical design

🛠️ Technologies Used

Python 3

Tkinter – Used to build the graphical user interface

Pillow (PIL) – Used for image loading and resizing

Datetime Module – Used to store order date and time

OS Module – Used for file handling and management
